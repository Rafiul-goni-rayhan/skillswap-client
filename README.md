# SkillSwap - Freelancing Marketplace Platform

A modern, full-stack freelancing platform built with Next.js 16 (App Router), HeroUI, Node.js, Express, and MongoDB.

## Features
- **Secure Authentication:** Cookie-based JWT login/registration with role selection (Client/Freelancer).
- **Dynamic Navbar:** Synchronized global navigation bar with active user state from localStorage.
- **Client Pipeline:** Seamless task creation with modal forms and real-time proposal management (Accept Bid workflow).
- **Freelancer Pipeline:** Dynamic task marketplace browser, targeted bidding system, and real-time earnings tracker.
- **Aesthetics:** Dark-themed responsive design featuring modern glassmorphism layouts.

## Tech Stack
- **Frontend:** Next.js, React, Tailwind CSS, HeroUI
- **Backend:** Node.js, Express, MongoDB, JSON Web Tokens (JWT)
